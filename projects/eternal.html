Eternal - Smart Ambulance Booking System

  • Tech Stack: EJS(Embedded JavaScript),Express.JS, NodeJS, Python, React, Stripe , Google API, Streamlit
  • Github URL: Project Link

Eternal is a smart ambulance booking system designed to improve emergency healthcare access in India. The system ensures quick ambulance dispatch, real-time tracking, seamless payment, and integration with emergency helplines.

Features:

  • User Registration & Booking: Simple interface for emergency ambulance booking (24/7).
  • Real-time Tracking: GPS-based tracking of ambulance location and estimated arrival.
  • Payment Integration: Secure transactions via UPI, credit/debit cards, and wallets.
  • Hospital Finder: Displays nearest hospitals with optimized routes.
  • Driver Registration & Availability: Real-time status update for ambulance drivers.
  • Emergency Helpline Integration: National and state-level emergency number integration.
  • Feedback & Ratings: User reviews to improve service quality.

Bonus Features:

  • Push Notifications: Real-time alerts on ambulance dispatch and arrival.
  • Multilingual Support: Regional language support for better accessibility.
  • AI-based Predictive Response: Smart response time prediction based on traffic data.
  • Pre-registration of Health Data: Store medical history for faster treatment.
  • Government & Insurance Integration: Emergency coverage and subsidy support.
