Golden Finger Hotels

  • Tech Stack:TypeScript,Next.js, PostgreSQL, React, Node.js, Firebase
  • Website URL: Live Demo
  • Github URL: Project Link

Golden Finger Hotels is a comprehensive hotel management system that enables seamless room booking, staff management, and revenue tracking.

Features:

  • Room Booking: Online reservation system with real-time availability updates.
  • Customer Dashboard: Personalized user accounts for booking history and preferences.
  • Payment Integration: Secure transactions via UPI, credit/debit cards, and wallets.
  • Staff Management: Admin panel for employee scheduling and task assignments.
  • Reviews & Ratings: Guest feedback system to enhance service quality.
